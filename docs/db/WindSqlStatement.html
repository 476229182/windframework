Class WindSqlStatement

Class Overview
Class Overview | Method Summary | Methods Detail

sql语句处理类,该类封装了基础的sql处理方法

实现了基础的,数据绑定,参数绑定以及基础查询接口. 通过调用'WindConnection'的'createStatement()'方法,可以获得一个statement对象.
