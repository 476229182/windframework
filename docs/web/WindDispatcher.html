  void dispatchWithAction( WindForward $forward, WindRouter $router, boolean $display  )

重定向请求到新的action操作

该种重定向类型,是中断当前的请求执行过程,开启另外的action操作处理.是在一次请求内部进行重定向, 所以之前的一些处理的结果变量,在重定向后是会继续存在,并可通过forward变量进行访问的.也就是不仅仅是过程的重定向, 也是状态的重定向.
