Class WindForwardException

Class Overview
Class Overview | Inherited Properties, Constants, and Methods | Method Summary | Methods Detail | Constants

用于跳转的异常类型

同WindActionException相同都是用于程序流程控制,当程序需要forward操作时,抛出该异常类型并设置一个forward对象给它. 系统捕获该异常后,获得forward对象,根据forward携带的信息进行后续处理
