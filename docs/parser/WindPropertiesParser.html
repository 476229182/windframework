Class WindPropertiesParser

Class Overview | Method Summary | Methods Detail | Constants Summary | Constants Detail

properties格式文件解析

properties文件中的注释为单行注释以#标记 同时该文件也允许配置节名称，比如：

  1.  [test]
  2.  path=index.php
  3.  address.zipcode=10000
  4.  address.show=true //true将会被解析为boolean类型的true，false也将被解析成boolean类型的false
则上述的格式会解析成如下数组:
  1.  array(
  2.  'test' => array(
  3.          'path' => 'index.php',
  4.          'address' => array(
  5.              'zipcode' => '10000',
  6.              'show' => true
  7.       )
  8.   )
  9. )
